## Crazy Arcade (크레이지 아케이드) 모작
C++, DirectX11, VisualStudio2022 를 이용한 [크레이지 아케이드](https://ca.nexon.com/Home/Index) 모작입니다.
<br/><br/>

## 설치
#### git clone
```
git clone https://github.com/rla5233/DirectX_CrazyArcade.git
```

#### 솔루션 실행
DirectX_CrazyArcade.sln 을 실행해 주세요.
